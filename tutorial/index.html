Tutorial

This is a tutorial to understand how to use the html-template changer. In the read me file you ac find more information. The tutorial consist of two html files index.html and contact.html ,

Modifying the footer.

Let us start with a simple change, the footer.
  1. Select Inspect html file .
  2. Enter index.mthl as the file to inspect it.
  3. Select Print index.html , to get an overview of the file, also to be sure that the right file was selected. As you can see the year in the footer is wrong. You can open both html file and you will see that the year is wrong in both files.
  4. Select Select tag and enter footer as the tag to change.
  5. Select Show selector to display the selctor. The script should display Your selection is .footer. if the script displays something different, select Reset selection and follow the previous step.
  6. Select Show selected content to display the the the content of the footer. The script should display <footer> 2009 </footer>
  7. Since the only content of the footer is a string, select Select attribute and enter string .
  8. Select Show selected content to display the the the content that was selected. The script should display 2009 if the script displays something different, select Reset selection and follow the previous steps.
  9. Select Change content our selection should be soup.footer.string if that is correct, then proceed.
  10. Now, enter the year 2022
  11. Select Print index.html to check that the footer now says 2022.
  12. This is just a preview of the changes, to make these changes in the html files, return to main menu
  13. In the main menu, select Modify template
  14. Load the webpages, the webpages must be listed in advance in the file webpages.csv
  15. Select Modify webpages , the script should display
    modifying: index.html
    modifying: contact.html .
  16. Now you can exit and check that the footer has been changed in both files.

Modifying the News text.

Let us continue with the text in the news section. The news talks about a dog found in the Moon, but you know dogs cannot live in the moon but cats can. Let us change dog to cat.
  1. Select Inspect html file .
  2. Enter index.mthl as the file to inspect it.
  3. Select Print index.html , to get an overview of the file, also to be sure that the right file was selected. As you can see the news are nested in a div with id "News_text".
  4. Select Select id and enter News_text as the id to change.
  5. Select Select tag and enter p as the tag to change.
  6. Select Show selector to display the selctor. The script should display Your selection is .find(id='News_text').p. if the script displays something different, select Reset selection and follow the previous steps.
  7. Select Show selected content to display the the the content of our selectionr. The script should display <p> A dog was found a live in the Moon </p>
  8. Since the only content of our is a string, select Select attribute and enter string .
  9. Select Show selected content to display the the the content that was selected. The script should display A dog was found a live in the Moon if the script displays something different, select Reset selection and follow the previous steps.
  10. Select Change content our selection should be soup.find(id='News_text').p.string if that is correct, then proceed.
  11. Now, enter A cat was found living in the Moon
  12. Select Print index.html to check that the news are now about a cat.
  13. This is just a preview of the changes, to make these changes in the html files, return to main menu
  14. In the main menu, select Modify template
  15. Load the webpages, the webpages must be listed in advance in the file webpages.csv
  16. Select Modify webpages , the script should display
    modifying: index.html
    modifying: contact.html .
  17. Now you can exit and check that the news has been changed in both files.

Modifying the picture.

Let us continue with the picture of the webpage. Now the news talks about a cat found in the Moon, but the picture is about a dog and it doesn't load. Let us change it to cat one.
  1. Select Inspect html file .
  2. Enter index.mthl as the file to inspect it.
  3. Select Print index.html , to get an overview of the file, also to be sure that the right file was selected. As you can see the picture is nested in a div with id "main_div".
  4. Select Select id and enter main_div as the id to change.
  5. Select Select tag and enter figure as the tag to change.
  6. Since the picture is an img tag nested in the figure tag, select Select tag and enter img as the tag to change.
  7. Select Show selector to display the selctor. The script should display Your selection is .find(id='main_div').figure.img if the script displays something different, select Reset selection and follow the previous steps.
  8. Select Show selected content to display the the the content of our selectionr. The script should display <img alt="A dog picture" src=".dog.jpg"/>
  9. Since we want to change the picture, select Select attribute and enter src .
  10. Select Show selected content to display the the the content that was selected. The script should display .dog.jpg if the script displays something different, select Reset selection and follow the previous steps.
  11. Select Change content our selection should be soup.find(id='main_div').figure.img['src'] if that is correct, then proceed.
  12. Now, enter cat.jpg
  13. Select Print index.html to check that the picture is cat.jpg.
  14. This is just a preview of the changes, to make these changes in the html files, return to main menu
  15. In the main menu, select Modify template
  16. Load the webpages, the webpages must be listed in advance in the file webpages.csv
  17. Select Modify webpages , the script should display
